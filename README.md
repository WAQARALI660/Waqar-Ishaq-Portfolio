A modern, responsive AI Developer portfolio built with HTML, CSS, and JavaScript. Features dark theme, animations, and glowing effects to showcase AI/ML skills and projects.

Live Demo: [https://waqarali660.github.io/portfolio/](https://waqarali660.github.io/portfolio/)

## Features

- **Modern Dark Design** - Eye-catching dark theme with gradient accents
- **Responsive Layout** - Works perfectly on mobile, tablet, and desktop
- **Interactive Animations** - Smooth scrolling, hover effects, and glowing elements
- **AI/ML Skills Showcase** - Comprehensive display of technical skills
- **Project Portfolio** - Gallery of completed AI/ML projects
- **Direct Contact** - Easy ways for recruiters to connect

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Icons:** Font Awesome 6
- **Fonts:** Google Fonts (Poppins)
- **Hosting:** GitHub Pages
- **Design:** Custom CSS with CSS Grid & Flexbox
